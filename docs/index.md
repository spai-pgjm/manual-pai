## Comandos

* `mkdocs new [dir-name]` - Cria um novo projeto.
* `mkdocs serve` - Inicia o carregamento local do servidor de aplicação.
* `mkdocs build` - Constrói o build do projeto.
* `mkdocs -h` - Mostra mensagem de ajuda e sai.

## Leiaute do projeto
